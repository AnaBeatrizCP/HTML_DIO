° sup

(Voltar)

Para criar texto em sobrescrito, como números elevados, símbolos matemáticos e notas de rodapé.

° blockquote

(Voltar)

Usado para formatar citações longas ou blocos de texto citados, normalmente recuando o texto e aplicando um estilo de fonte diferente.

° font

(Voltar)

Usada para definir o estilo de fonte, tamanho e cor do texto.

° mark

(Voltar)

Esta tag é usada para destacar partes do texto, muitas vezes aplicando um fundo amarelo ou outra formatação visual para chamar a atenção para o conteúdo.

° del

(Voltar)

Usado para marcar texto como deletado ou removido. Normalmente, ele exibe o texto com uma linha através dele.

° abbr

(Voltar)

Usado para marcar acrônimos ou abreviações, permitindo que você forneça uma descrição completa quando o usuário passa o mouse sobre o termo abreviado.

° strong

(Voltar)

Esta tag é usada para dar ênfase forte ao texto, geralmente fazendo com que ele apareça em negrito. Deve ser usado para ênfase semântica, como destacar palavras-chave importantes.

° u

(Voltar)

É usada para sublinhar o texto, indicando que o texto é uma referência ou link.

° i

(Voltar)

usada para aplicar ênfase ou estilo de itálico ao texto

° sub

(Voltar)

Usado para criar texto em subscrito

° h1, h2, h3, h4, h5, h6

(Voltar)

Essas tags são usadas para criar cabeçalhos e subtítulos em ordem de importância, do &lh1> (mais importante) ao &lh6> (menos importante).

° li

(Voltar)

usada para criar itens em uma lista, seja uma lista não ordenada (&lul>) ou uma lista ordenada (&lol>). Os itens da lista são marcados por essa tag.

° hr

(Voltar)

Cria uma linha horizontal que é usada para separar seções de conteúdo na página.

° small

(Voltar)

É usada para diminuir o tamanho do texto e é frequentemente usada para texto de rodapé ou notas de rodapé.

° a

(Voltar)

° p

(Voltar)

é usada para criar parágrafos de texto no corpo do documento HTML. Cada parágrafo é separado por esta tag.
